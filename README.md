# AI Smart Time Manager

A smart task scheduling web application built using Flask and SQLite.

## Features
- Add tasks with title, description, and deadline
- Automatic priority detection
- Overdue task detection
- Dashboard with analytics chart
- Calendar view with drag and drop
- Task search and filtering
- Responsive UI

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap
- JavaScript
- FullCalendar.js
- Chart.js

## How to Run
1. Install Python
2. Install Flask
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000
